# euryx
A collection of Python 🐍 scripts made just for fun and learning


## Requirements


- Python 3.10.5
- git 2.36.1


## Installation


0. Clone the repository ``git clone https://github.com/tricantivu/euryx.git``.
1. Open the RC file for your shell (e.g ``.bashrc``, ``.zshrc``, etc.) with your editor of choice.
2. Add the following line ``PATH=${absolutePathToRepository}/src:${PATH}`` or modify an existent assignment to match the former.
3. Save changes and quit the file.
4. Restart your shell.
